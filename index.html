NeuroRANO · Ajuda & Sobre

O que é o NeuroRANO?

NeuroRANO é uma aplicação web open-source de apoio à decisão clínica que operacionaliza os critérios RANO 2.0 (Wen et al., JCO 2023) para avaliação da resposta ao tratamento em gliomas do adulto. Não é um dispositivo médico e não substitui o julgamento clínico do neurorradiologista ou neuro-oncologista.

O que faz

  • Cálculo automático de SPD (2D) e volume (3D) das lesões-alvo
  • Comparação automática com estudo baseline ou nadir
  • Aplicação determinística das regras RANO 2.0 com justificação visível
  • Caracterização morfológica e funcional (ADC, perfusão, T2, padrão de realce)
  • Geração de relatório estruturado (PDF, CSV, copy-paste)
  • Multilingue: PT · EN · ES

Limitações

  • Validado apenas para gliomas do adulto (não aplicar a metástases, linfoma, tumores pediátricos)
  • Não processa imagens DICOM — as medições têm de ser introduzidas manualmente
  • Não armazena dados — toda a informação é descartada ao recarregar a página

Fluxo de uso

  1. Etapa 1 — Contexto clínico: escolhe o objectivo (resposta vs progressão), diagnóstico WHO 2021, fase, modo (2D/3D) e factores confundidores.
  2. Etapa 2 — Medições: introduz dimensões das lesões. SPD/volume é calculado automaticamente. Marca a imagem-chave do PACS.
  3. Etapa 3 — Comparação: introduz as dimensões do estudo de referência (baseline ou nadir). A variação percentual aparece automaticamente.
  4. Etapa 4 — Critérios extra: novas lesões, leptomeníngea, corticoides, clínica e caracterização morfológica/funcional.
  5. Etapa 5 — Resultado: categoria RANO 2.0 sugerida, critérios cumpridos, regras aplicadas e relatório estruturado pronto a copiar/exportar.

Atalhos úteis

  • Exemplos A–D pré-carregam casos típicos para experimentação
  • O painel à direita mostra sempre a classificação corrente
  • Todos os campos com ? têm explicações detalhadas
  • Esc fecha esta janela

Perguntas frequentes

A app guarda os dados que introduzo?
Não. Tudo é processado localmente no navegador e descartado ao recarregar a página. Não há servidor.
Posso usar para tomar decisões clínicas?
É uma ferramenta de apoio. A interpretação final é sempre do clínico responsável. Não está aprovada como dispositivo médico.
Porque é que a app sugere SD se eu acho que é PD?
Verifica: (1) redução de corticoides abaixo do nadir; (2) Δ vs nadir < 25% (2D) ou < 40% (3D); (3) ausência de novas lesões; (4) ausência de progressão não-alvo. O painel "Justificação" mostra todas as regras aplicadas.
Quando devo escolher 2D vs 3D?
Por defeito, RANO 2.0 baseia-se em 2D (SPD). 3D (volumétrico) é alternativa para ensaios clínicos quando há software de segmentação validado. Limiares mais restritivos (PR −65%, PD +40%).
Funciona offline?
Sim. Depois de carregar uma vez, todo o cálculo é local. Podes guardar o ficheiro HTML e usar sem internet.
Posso contribuir / sugerir melhorias?
Sim — código aberto no GitHub. Abre uma issue ou pull request.
🛡
A app não recolhe, transmite nem armazena dados de doentes. Todo o processamento ocorre no navegador. Não há base de dados nem cookies de tracking.

Privacidade (RGPD)

  • Nenhum identificador directo de doente deve ser introduzido (sem nome, MRN, nº de processo, data de nascimento).
  • O campo "ID interno do caso" destina-se a um código arbitrário escolhido pelo utilizador.
  • O relatório exportado (PDF/CSV) fica no dispositivo do utilizador — a responsabilidade pelo tratamento subsequente é do utilizador / instituição.

Uso de imagens em publicações ou ensino público

  • Imagens devem ser integralmente anonimizadas (DICOM cleaned, sem texto na imagem, defacing se aplicável).
  • Recomenda-se parecer da Comissão de Ética da instituição.
  • Alternativa: bancos abertos — Radiopaedia, TCIA, BraTS — respeitando licenças (CC BY-NC-SA, etc.).

Responsabilidade clínica

A classificação é meramente indicativa. A interpretação radiológica e a decisão terapêutica são da responsabilidade exclusiva do clínico assistente no contexto do tumor board multidisciplinar.
